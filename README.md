# cuda-energy

**ATP budgets, circadian rhythms, and the thermodynamics of agent computation.**

> Every computation has a cost. Every action consumes energy. Rest generates it.

## Energy Model

### ATP (Adenosine Triphosphate)
The universal energy currency. Generated by rest, consumed by every other operation.

| Operation | ATP Cost |
|-----------|----------|
| Perception | 0.5 |
| Arithmetic | 0.1 |
| Deliberation | 2.0 |
| Communication | 1.0 |
| Memory access | 0.2 |
| Instinct activation | 0.8 |
| Rest (generates) | -1.0 |
| Apoptosis | 0.0 (terminal) |

### Circadian Rhythm
Cosine modulation of instinct strength based on time of day:
- Navigate peaks at noon (12:00)
- Communicate peaks at 14:00
- Perceive peaks at 10:00
- Rest peaks at 02:00
- Floor at 0.1 (no instinct ever goes completely silent)

### Epigenetic Memory
Experience modifies instinct weights permanently. Repeated use of an instinct in a specific context strengthens it. This is the "learned" layer on top of inherited instincts.

### Apoptosis Protocol
Graceful self-termination when fitness drops:
- Energy threshold: 0.05
- Fitness threshold: 0.1
- Patience: 10 consecutive low ticks before trigger
- Releases resources back to fleet

## Usage

```rust
use cuda_energy::{EnergyBudget, CircadianRhythm, ApoptosisProtocol};

let mut budget = EnergyBudget::new(100.0);
budget.consume(cuda_energy::Operation::Deliberation);  // -2.0 ATP
budget.consume(cuda_energy::Operation::Rest);           // +1.0 ATP

let rhythm = CircadianRhythm::new();
let nav_strength = rhythm.modulate(Instinct::Navigate, hour=12);  // ~0.95
let rest_strength = rhythm.modulate(Instinct::Rest, hour=2);     // ~0.95
```
